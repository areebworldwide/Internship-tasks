## Description

A basic chatbot that responds to greetings and simple questions using predefined rules.

## Features

- Responds to: "hello", "how are you", "bye", "thank you"
- Case-insensitive matching
- Continuous conversation until user exits

## How to Run

```bash
python chatbot.py
```

## Usage Example

```
You: hello
Bot: Hi! Welcome to my chatbot!

You: how are you
Bot: I'm fine, thanks! How are you doing?

You: bye
Bot: Goodbye! Have a nice day!
```

## Key Concepts Used

- Functions (`chatbot_response()`, `main()`)
- if-elif-else statements
- while loops
- Input/Output operations
- String methods (`.lower()`)
